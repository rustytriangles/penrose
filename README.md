### Commands
* D - Use Dart tile
* K - Use Kite tile
* F - Use Fat Rhombus tile
* S - Use Skinny Rhombus tile
* A - Toggle arcs
* B - Toggle Ammann bars
* C - Clear
* E - Toggle edges
* X - Enable debugging
* U - Remove last tile
* Up - Scale up
* Down - Scale down
* Left - Rotate left
* Right - Rotate right

### Modules used

* [nannou](https://nannou.cc/) Nannou framework
